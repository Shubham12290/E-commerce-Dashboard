📊 E-Commerce Business Intelligence & Predictive Analytics Platform

🚀 Project Overview

This project is an end-to-end Data Analytics Solution built for an e-commerce company ShopSmart, which sells consumer electronics.

The goal is to transform raw business data into meaningful insights, predictions, and interactive dashboards to support better decision-making.

🎯 Objectives
Analyze sales data to extract business insights
Perform statistical analysis and hypothesis testing
Build visualizations for better understanding
Develop a simple machine learning model
Create an interactive dashboard for stakeholders

🛠️ Tech Stack
Excel – Data cleaning, formulas, pivot tables, dashboard
SQL – Database creation & querying
Python – Data analysis & visualization
Pandas
NumPy
Matplotlib / Seaborn
Machine Learning – Basic predictive model
Power BI / Tableau – Interactive dashboard
